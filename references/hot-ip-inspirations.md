# 热门游戏与动漫灵感参考

记录时间：2026-03-26

## 用法

当用户说“想更火一点”“加点热门游戏/动漫感觉”“想借现在流行题材”时，读取这份文档。

默认规则：
- 借的是环境、气质、世界观、色彩和镜头氛围
- 不默认替换成别的作品角色
- 默认不使用《原神》《崩坏：星穹铁道》《绝区零》作为参考来源
- 除非用户明确点名 crossover，否则优先写成“某种热门题材气质”

## 时效检查

- 每次参考这份文档前，先检查顶部的 `记录时间`
- 如果当前日期距离 `记录时间` 已超过两个月，不直接把这里的“热门”判断当作最新事实
- 超过两个月时，先请求更新文档，再继续使用“热门 IP 灵感”能力

## 热门游戏方向

### 鸣潮

适合借鉴：
- 都市废墟与未来感并存
- 风感、海边、灰蓝调、冷感建筑
- 末世后城市的清冷空旷感

适合脑暴：
- 海边城市天台发呆
- 冷感街区里的安静眨眼停顿
- 未来废墟里拿手机看消息

### 怪物猎人：荒野

适合借鉴：
- 荒野营地
- 大地图自然场景
- 篝火、木箱、装备台、广阔地形
- 野外临时据点气质

适合脑暴：
- 营地帐篷前抱着热饮发呆
- 装备台边认真整理小道具
- 荒野风口里回头轻轻挥手

## 热门动漫方向

### 灵笼 第二季

适合借鉴：
- 国创科幻压迫感
- 高反差文明空间
- 冷硬装置、巨大结构、末世聚落
- 压抑和烟火气并存

适合脑暴：
- 巨型机械通道里停下回头
- 末世聚落摊位前抱着小袋子发呆
- 高空平台边安静看远处再回头

### 葬送的芙莉莲 第二季

适合借鉴：
- 安静奇幻旅行感
- 森林、小镇、石路、旅店、黄昏
- 温柔空气感和轻哀愁

适合脑暴：
- 石板路边站着发呆
- 旅店窗边看外面的小表情
- 黄昏树林里慢慢回头

### 咒术回战

适合借鉴：
- 高张力现代都市异能感
- 校园、街区、昏暗走廊、天台
- 强烈压迫感和动作前摇

适合脑暴：
- 校园走廊尽头盯镜头
- 天台边鼓脸逞强卖萌
- 昏暗街口回头做个轻甜停顿

### 中国奇谭2

适合借鉴：
- 中式奇谭
- 民俗、寓言、山野、旧街巷
- 强风格化美术和中式想象力

适合脑暴：
- 老街灯笼下转头甜笑
- 山野路边的小跳步
- 中式小店门口抱着纸袋乖乖等人

### 哪吒2

适合借鉴：
- 高能神话少年感
- 东方神话空间
- 火焰、海浪、法器、夸张动势
- 热闹、张扬、情绪外放

适合脑暴：
- 神话街市里元气挥手
- 海边礁石上鼓脸回头卖萌
- 法器摊位前认真挑发光小道具

## 快速配方

如果用户只说“加点热门作品感觉”，优先套下面这些组合：

- 雨夜橱窗 + 鸣潮式冷感城市气质
- 校园窗边 + 咒术回战式现代校园异能氛围
- 黄昏旅店 + 葬送的芙莉莲式安静奇幻气质
- 老街灯笼 + 中国奇谭2式中式奇谭气质
- 末世平台 + 灵笼第二季式国创科幻压迫感

## 提醒

- 一个点子默认只借 1 到 2 个热门 IP 气质。
- 超过 2 个来源时，容易变成杂糅和拼贴。
- 优先借环境和氛围，不要把提示词写成“全都像一点”。
