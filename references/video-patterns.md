# 视频提示词结构模板

## 用法

优先选一个结构模板，不要把多个模板硬拼到同一条提示词里。

## 模板 1：萌系循环短片

适合：
- Q版萌化
- 轻动作
- 表情记忆点
- 可循环播放

结构：

```md
主体 + 单场景 + 一个轻动作起点 + 动作持续节奏 + 轻镜头跟随 + 柔和氛围 + 结尾回到开头附近
```

## 模板 2：情绪近景短片

适合：
- 靠表情出效果
- 靠眼神和轻动作传情绪
- 角色存在感强于场景

结构：

```md
主体近景 + 明确表情变化 + 极简动作 + 微推或轻摇镜头 + 质感和光线强化 + 停留收尾
```

## 模板 3：角色出场短片

适合：
- 角色亮相
- 突出服装、气质和第一眼记忆点
- 更有短视频开场感

结构：

```md
主体入画方式 + 视线或姿态变化 + 镜头跟入或轻推近 + 风格锚点 + 收尾定格
```

## 模板 4：轻剧情反应短片

适合：
- 一个动作引发一个反应
- 可爱、惊讶、害羞、得意这类单点情绪

结构：

```md
动作起因 + 角色反应 + 镜头跟住反应 + 情绪放大 + 结尾停留或轻回弹
```

## 槽位填写顺序

按这个顺序补全提示词：

1. 主体是谁
2. 场景在哪
3. 动作从哪里开始
4. 镜头怎么跟
5. 情绪是什么
6. 光线和质感是什么
7. 结尾如何收住
8. 负面约束是什么
