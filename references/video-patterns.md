# 视频提示词结构模板

## 用法

优先选一个结构模板，不要把多个模板硬拼到同一条提示词里。
默认用户会上传参考图，因此模板重心放在环境、动作、镜头和 15 秒节奏，而不是反复描述角色外形。

## 模板 1：参考图环境迁移短片

适合：
- 已有参考图
- 进入一个明确环境
- 强表情记忆点
- 有完整 15 秒节奏

结构：

```md
参考图角色 + 一个明确环境 + 与环境发生接触的动作 + 轻镜头或定镜 + 15 秒收尾停顿
```

## 模板 2：情绪近景短片

适合：
- 靠表情出效果
- 靠眼神和轻动作传情绪
- 角色存在感强于场景
- 适合 15 秒近景反应

结构：

```md
主体近景 + 明确表情变化 + 极简动作 + 微推或轻摇镜头 + 0-5/5-10/10-15 秒节奏 + 停留收尾
```

## 模板 3：角色出场短片

适合：
- 角色亮相
- 突出角色态度和第一眼记忆点
- 更有短视频开场感
- 适合从环境里出现

结构：

```md
主体入画方式 + 视线或姿态变化 + 镜头跟入或轻推近 + 环境锚点 + 收尾定格
```

## 模板 4：现实环境小剧场

适合：
- 便利店、教室、房间、咖啡店、街道
- 角色与现实道具互动
- 有轻反差感
- 适合 15 秒小梗

结构：

```md
参考图角色 + 现实场所锚点 + 一个小道具或动作 + 近景或半身镜头 + 表情反应 + 15 秒停顿收尾
```

## 模板 5：轻剧情反应短片

适合：
- 一个动作引发一个反应
- 无语、小生气、傲娇、嫌弃这类单点情绪

结构：

```md
动作起因 + 角色反应 + 镜头跟住反应 + 情绪放大 + 结尾停留或轻回弹
```

## 槽位填写顺序

按这个顺序补全提示词：

1. 参考图角色在什么环境
2. 场景锚点是什么
3. 动作从哪里开始
4. 镜头怎么跟
5. 情绪是什么
6. 光线和质感是什么
7. 0-5 秒发生什么
8. 5-10 秒发生什么
9. 10-15 秒如何收住
