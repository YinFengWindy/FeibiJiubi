# 菲比啾比题材母题池

## 说明

这份文档用于头脑风暴，不是热度榜单。
默认用户会上传参考图，所以题材发散重点放在“让这个小人进入不同环境，并在 15 秒里形成一个完整反应”。

## 常见母题

### 1. 现实教室 / 窗边吐槽

关键词：
- 靠窗课桌
- 教室光线
- 小人坐在桌边
- 无语或嘴硬反应
- 现实校园感

适合延展：
- 开学、上课、发呆、小纸条题材
- 很适合做现实校园反差
- 环境细节容易识别

### 2. 便利店 / 买东西小剧场

关键词：
- 货架
- 购物篮
- 冰柜或收银台
- 小人生气或吐槽

适合延展：
- 道具互动强
- 很容易出生活感
- 适合做短梗和反差

### 3. 房间 / 床边或桌边碎碎念

关键词：
- 房间暖光
- 床边或桌边
- 拿手机或抱枕
- 小人碎碎念

适合延展：
- 居家感强
- 适合夜晚或日常题材
- 可以做第一视角陪伴感

### 4. 咖啡店 / 窗边观察别人

关键词：
- 咖啡杯
- 靠窗座位
- 街景背景
- 小人嫌弃或观察感

适合延展：
- 都市氛围好做
- 适合静态表情戏
- 现实环境辨识度高

### 5. 二次元校园 / 樱花坡道

关键词：
- 动漫教室或校门口
- 樱花
- 校服感
- 小人傲娇或嘴硬

适合延展：
- 很适合二次元世界观
- 环境很容易一眼识别
- 可搭配轻风和轻停顿

### 6. 异世界商店 / 魔法柜台

关键词：
- 魔法道具
- 木柜台
- 发光瓶子
- 小人一脸无语地营业

适合延展：
- 场景新鲜感强
- 适合轻整活
- 现实感和幻想感反差大

### 7. 赛博街区 / 霓虹夜景

关键词：
- 霓虹灯
- 夜雨地面反光
- 街边屏幕
- 小人嘴硬又拽

适合延展：
- 环境冲击力强
- 适合短视频开场抓点
- 适合半写实融合

### 8. 小伙伴互动 / 环境里一起演

关键词：
- 小伙伴或小生物
- 一个明确环境
- 一方吐槽
- 一方装无辜

适合延展：
- 双主体小品
- 角色和环境一起服务梗
- 适合做连续系列

## 发散公式

用下面公式从母题生成新点子：

```md
母题 + 一个情绪中心 + 一个记忆表情 + 一个与环境接触的动作 + 一个场所锚点 + 一个 loop 结尾
```

## 默认推荐组合

如果用户没有指定方向，优先从这些组合开始：

- 教室窗边 + 无语 + 托腮 + 课桌边缘 + 表情复位
- 便利店货架前 + 小生气 + 拿饮料或购物篮 + 货架停顿 + 口型停顿
- 咖啡店窗边 + 嫌弃 + 盯外面看 + 杯子边缘 + 定格停顿
- 二次元校园 + 傲娇 + 转头甩一句话 + 樱花飘落 + 回弹收尾
- 赛博街区 + 嘴硬 + 拿手机看消息 + 霓虹路面反光 + 回头收尾

## 使用提醒

- 发散时优先保留圈层熟悉感，再做小变体。
- 优先做“同环境不同情绪”和“同表情不同环境”的变化。
- 如果用户要的是更火一点的方向，优先从现实教室、便利店、二次元校园、赛博街区四类开始。
