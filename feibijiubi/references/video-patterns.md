# 视频提示词结构模板

## 用法

优先选一个结构模板，不要把多个模板硬拼到同一条提示词里。
默认用户会上传参考图，因此模板重心放在动作、表情、镜头和 15 秒节奏，而不是剧情推进。

## 模板 1：动作卖点近景短片

适合：
- 靠表情和轻动作出效果
- 角色存在感强于场景
- 开头就想给一个清楚的动作钩子
- 适合 15 秒近景反应

结构：

```md
主体近景 + 一个明确主动作 + 一个表情变化 + 微推或轻摇镜头 + 0-5/5-10/10-15 秒节奏 + 停留收尾
```

## 模板 2：环境承托动作短片

适合：
- 已有参考图
- 环境只负责托住动作
- 角色会和桌边、窗台、货架、杯子等发生接触
- 有完整 15 秒节奏

结构：

```md
参考图角色 + 一个明确环境 + 与环境发生接触的动作 + 轻镜头或定镜 + 动作放大 + 15 秒收尾停顿
```

## 模板 3：出场动作钩子短片

适合：
- 角色亮相
- 突出第一眼动作记忆点
- 更有短视频开场感
- 适合从环境里出现

结构：

```md
主体入画方式 + 一个可爱出场动作 + 视线或姿态变化 + 镜头跟入或轻推近 + 环境锚点 + 收尾定格
```

## 模板 4：道具互动卖萌短片

适合：
- 想靠杯子、手机、购物篮、抱枕、小牌子等道具出效果
- 动作需要一个接触点才成立
- 适合半身镜头

结构：

```md
参考图角色 + 一个道具 + 一个连续互动动作 + 表情反应 + 半身机位 + 15 秒停顿收尾
```

## 模板 5：单反应 loop 短片

适合：
- 一个动作引发一个反应
- 可爱、害羞、小开心、小得意、甜酷这类单点情绪
- 适合 loop 收尾

结构：

```md
动作起点 + 表情反应 + 镜头跟住反应 + 情绪放大 + 结尾停留或轻回弹
```

## 槽位填写顺序

按这个顺序补全提示词：

1. 主动作是什么
2. 表情反应是什么
3. 环境怎样托住这个动作
4. 场景锚点是什么
5. 有什么接触关系或道具
6. 镜头怎么跟
7. 光线和质感是什么
8. 0-5 秒发生什么
9. 5-10 秒发生什么
10. 10-15 秒如何收住
