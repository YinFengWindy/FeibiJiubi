# 菲比啾比动作母题池

## 说明

这份文档用于头脑风暴，不是排序清单。
默认用户会上传参考图，所以发散重点放在“让这个小人用一个可爱动作在 15 秒里形成记忆点”，不是先编剧情。

## 常见动作母题

### 1. 歪头靠近镜头

关键词：
- 轻轻歪头
- 眼神看镜头
- 微微前探
- 甜笑或装无辜

适合延展：
- 近景表情戏
- 第一眼抓人
- 很适合做 loop

### 2. 双手捧脸或托腮

关键词：
- 双手托脸
- 桌边、窗边、柜台边
- 轻轻晃动
- 软萌发呆或甜甜眨眼

适合延展：
- 教室、咖啡店、房间窗边
- 适合静态表情戏
- 很容易形成可爱截图点

### 3. 抱杯子或抱枕蹭蹭

关键词：
- 抱住道具
- 身体轻轻靠上去
- 表情软下来
- 有陪伴感

适合延展：
- 房间、咖啡店、便利店
- 道具互动强
- 很适合做轻甜日常感

### 4. 小跳步或小跑转头

关键词：
- 小步伐
- 转身回头
- 元气挥手或甜笑
- 动作节奏更明显

适合延展：
- 校园、街道、樱花坡道、赛博街区
- 开场抓点强
- 适合半身到中景

### 5. 鼓脸撒娇或捂脸害羞

关键词：
- 鼓脸
- 轻皱眉
- 捂嘴或捂脸
- 眼神闪躲

适合延展：
- 近景反应
- 容易出表情包感
- 适合弱剧情强反应

### 6. 挥手、比心、招呼

关键词：
- 单手挥手
- 双手比心
- 贴近镜头招呼
- 收尾定格

适合延展：
- 短视频开场或结尾
- 传播感强
- 很适合做系列化动作

### 7. 拿手机或电话卖萌

关键词：
- 拿手机看消息
- 拿电话贴脸
- 轻轻晃动
- 甜笑或小得意

适合延展：
- 房间、街道、赛博环境
- 现实生活感强
- 道具让动作更好理解

## 发散公式

用下面公式从母题生成新点子：

```md
动作母题 + 一个可爱情绪中心 + 一个表情变化 + 一个环境接触点 + 一个机位逻辑 + 一个 loop 结尾
```

## 默认推荐组合

如果用户没有指定方向，优先从这些组合开始：

- 歪头靠近镜头 + 软萌 + 甜笑 + 教室窗边 + 微推近 + 眨眼复位
- 双手托脸 + 小开心 + 轻轻晃动 + 咖啡店窗边 + 近景定镜 + 甜笑定格
- 抱杯子蹭蹭 + 轻甜 + 眼神发亮 + 咖啡店或房间 + 半身轻推 + 回弹收尾
- 小跳步转头 + 元气 + 挥手 + 二次元校园或街道 + 跟拍 + 回头眨眼收尾
- 拿手机卖萌 + 小得意 + 抿嘴笑 + 赛博街区或房间 + 半身轻摇 + 定格收尾

## 使用提醒

- 发散时优先保留动作辨识度，再决定环境细节。
- 优先做“同动作不同环境”和“同环境不同动作”的变化。
- 默认先从可爱、卖萌、轻甜路线发散；只有用户明确要求时，再转向吐槽、嫌弃、嘴硬等偏负面风格。
- 如果用户要的是更抓人、更直观的方向，优先从歪头靠近镜头、抱道具互动、小跳步转头、挥手比心四类开始。
